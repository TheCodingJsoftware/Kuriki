# Kuriki

Organize Curriculum Outcomes for Teachers.

## Install

First get firebase CLI:

    npm install -g firebase-tools

1. Log into Firebase using your Google account by running the following command:

    ```firebase login```

2. First run:

    ```npm install```

3. Then run dev:

    ```npm run dev```

4. To deply:

    ```npm run deploy```
